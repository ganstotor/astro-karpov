# Static vs Dynamic Websites

Understanding this difference helps you scale correctly.

## Static Websites

Generated at build time.

Pros:

- Very fast
- Cheap hosting
- Secure

Cons:

- Needs rebuild for updates

Best for:

- Blogs
- Marketing pages

## Dynamic Websites

Generated per request.

Pros:

- Real-time data
- Personalization
- Dashboards

Cons:

- Higher server cost
- More complexity

## Hybrid Approach

Modern frameworks allow mixing both.

Example:

- Static blog
- Dynamic user dashboard

---

Choose based on:

- Project complexity
- Budget
- SEO needs
