# Why Modern Websites Need Speed

Website speed is no longer optional. Users expect pages to load instantly, and search engines reward fast websites.

## First Impression Matters

Studies show that users leave a website if it takes longer than **3 seconds** to load.

Slow websites cause:

- Higher bounce rates
- Lower conversions
- Poor SEO rankings

## Google Ranking Factor

Google officially includes **Core Web Vitals** as ranking signals:

- Largest Contentful Paint (LCP)
- First Input Delay (FID)
- Cumulative Layout Shift (CLS)

## How to Improve Speed

- Use static site generators like Astro
- Optimize images
- Reduce JavaScript
- Use CDN delivery

## Conclusion

Fast websites win users and search rankings. Performance should be part of your design process from day one.
