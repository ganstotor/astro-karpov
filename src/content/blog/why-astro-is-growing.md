# Why Astro Is Growing So Fast

Astro focuses on one thing: **shipping less JavaScript**.

## Zero JS by Default

Astro sends no JS unless needed.

That means:

- Faster load
- Better SEO
- Smaller bundles

## Island Architecture

Interactive parts load independently.

Example:

- Static blog content
- React search component

Only search loads JS.

## Framework Agnostic

You can mix:

- React
- Vue
- Solid
- Svelte

Inside one project.

## Perfect for Content Sites

If you're building:

- Blogs
- Marketing sites
- Documentation

Astro is often better than Next.js.

---

### When NOT to Use Astro

If you're building:

- Large dashboard apps
- Heavy client logic apps

Next.js may be better.

---

Astro is simple. And simplicity wins.
